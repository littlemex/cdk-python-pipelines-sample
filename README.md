# cdk-python-pipelines-sample

```bash
source .venv/bin/activate
sudo .venv/bin/python3 -m pip install -r requirements.txt
cdk bootstrap aws://{accountid}/us-east-2
sudo .venv/bin/python3 -m pip install aws-cdk.aws_codecommit
sudo .venv/bin/python3 -m pip install aws-cdk.aws_codepipeline aws-cdk.aws_codepipeline-actions aws-cdk_pipelines
sudo .venv/bin/python3 -m pip install aws-cdk.aws_secretsmanager
```
